WP-Ajax-CommentsTemplate

Este plugin Wordpress sobscreve o template padrão de comentários, através de filtros permitindo adicionar validação de tamanho do comentário, bem como definir o estilo do formulário de envio e listagem de comentários. Utiliza PHP, AJAX e jQuery. Está publicado em http://wordpress.org/plugins/wp-ajax-comments-template

Download this project as a .zip file Download this project as a tar.gz file

Funcionalidades

Turbine seus comentários do Wordpress sem usar APIs de terceiros. Com esse plugin você consegue validar a quantidade mínima a máxima de caracteres, sem atualizar a página completamente. 

1. Configurar a forma de exibição dos labels dos campos
2. Quantidade Máxima de caracteres permitidos
3. Quantidade Mínima de caracteres permitidos
